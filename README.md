# mini-quiz-kajetan-szary

Projekt zawiera pewnego rodzaju quiz który po uruchoeminia zawiera 5 pytan z wyborami a,b,c,d jest tylko jedna poprawna odpowiedz oraz dodtakowo mozna resetowac quiz

 Otwórz projekt w Android Studio
 Uruchom na emulatorze (Pixel/Android 10+)
 Kliknij ROZPOCZNIJ QUIZ
